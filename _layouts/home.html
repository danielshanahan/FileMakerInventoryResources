Modifiable Data Model - $39

No need to start your inventory design from scratch! The modifiable data model allows you to edit, add, or delete the structural model of your database before you start to build. This data model comes in Visio, Omnigraffle, and SVG formats.

The SVG file can be opened and modified in many PC and Mac drawing applications. The Microsoft Visio and Omnigraffle are required for the other two formats.

Layout Mockups - $112

Drawing the layouts prior to creating them in FileMaker Pro saves time. Mockup tools help quickly create a visualization of the database layouts and provide a great deal of flexibility to quickly modify and test new ideas.

Mockups for Company, Contact, Item, Pick, Purchase Order, Receipt, Sales Order, Transaction, and Transfer. The collection includes 70 Files.

Layout mockups are created in Balsamiq Mockups and include the original file as well as PDF, JPG, and JSON.

B.O.M. Tree Pro - $199

The Bill of Materials tree allows users to see all levels of required materials for a manufactured part. The Pro version includes

  • Quantity Requested
  • Lead Times (for the item and another lead time for its WIP/Raw Materials)
  • Neet to Fulfill (NTF) quantity
  • Substitutions within the tree for a single line or all instances of a particular item
  • Save BOM trees
